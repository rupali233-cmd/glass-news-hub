# Glass News Hub 📰

Glass News Hub is a modern news website that displays real-time news using the NewsAPI.
The website features a glassmorphism user interface, category-based news filtering, search functionality, and an interactive chatbot.


## 🚀 Features

* 📰 Latest news using NewsAPI
* 📂 Category-based news filtering
* 🔎 News search functionality
* ⭐ Bookmark articles
* 🔊 Voice news reader
* 🤖 Simple chatbot assistant
* ✨ Glassmorphism UI design
* 📱 Responsive layout

## 🛠️ Technologies Used

* HTML
* CSS
* JavaScript
* NewsAPI
